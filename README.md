# :globe_with_meridians: App URL
https://card-game-drab-ten.vercel.app/

# :syringe: Install
```bash
git clone https://github.com/Recursion-Group-C/card-game.git
npm install
npm run dev
```
:point_right: http://localhost:3000

# :speech_balloon: Usage
1. ホーム画面からゲームを選択します
1. Bet画面で賭金を決めます
1. 各ゲームでハウスに勝ち、お金を増やしましょう
